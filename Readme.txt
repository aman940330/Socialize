1. Install Xampp to run this code on local server. 
2. In phpMyAdmin of local host create the database using the datbase-generation-code. 
3. Save the project in htdocs by using Xampp Explorer
4. Go to browser and type localhost/Socialize in the URL.
5. Login window appears. 
6. Click Sign-up link to get registration window. 
7. Register and then login to start using the system.
